## pos-seed

> 基于 vue 2.x、vue-cli 3.x

### 技术站
```
0. html: pug 模板引擎
1. css: stylus 预处理器
2. javascript
3. UI: vue-beauty 
```

### 安装依赖
```
yarn install  # 必须使用
```

### 运行
```
yarn dev
```

### 打包产出
```
yarn run build
```

### 代码检测
```
yarn lint
```

### 测试
```
yarn run test
```

### 注意 :point_left: :point_left: :point_left:
1. 必须使用 [yarn](https://yarnpkg.com/zh-Hans/) 依赖管理工具 :sunglasses:
